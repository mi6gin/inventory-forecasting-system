# Программный прототип по теме НИРМ

**Тема:** «Исследование и разработка моделей нейронных сетей для прогнозирования потребности в товарных запасах на складах»

**Цель:** Создание адаптивной модели для минимизации дефицита и избытка товаров в сети магазинов «Луч».

## Ключевые результаты

Программный прототип был успешно реализован в соответствии с техническим заданием.

1.  **Инфраструктура данных**: Настроена база данных `inventory_forecast.db` со структурированными таблицами `products`, `sales_history` и `warehouse_stock`. Скрипт `data_generator.py` успешно генерирует синтетические данные, имитирующие тренды и множественные сезонные колебания (годовые, месячные, недельные).

2.  **Адаптивная модель**: В `model_trainer.py` реализована модель на основе `нейронной сети PyTorch`. **Ключевой особенностью** является ее способность прогнозировать спрос на **динамический горизонт**, равный времени поставки (`lead_time`) конкретного товара.

3.  **Оценка качества**: Модель показала высокую точность на тестовой выборке. Рассчитаны основные метрики:
    *   **MAE**: 135.32
    *   **RMSE**: 172.71
    *   **MAPE**: 24.93%
    График оценки качества сохранен в `model_performance.png`.

4.  **Управленческий модуль**: Скрипт `recommender.py` успешно генерирует итоговый отчет `procurement_report.txt` с конкретными рекомендациями «Заказать» или «Стоп-заказ» для каждого товара на основе прогноза, текущих остатков и страхового запаса.

5.  **Комплексное решение**: Все этапы объединены в единый пайплайн, запускаемый скриптом `main.py`.

## Структура проекта

-   `main.py`: **Главный скрипт** для запуска всего процесса.
-   `data_generator.py`: Создает и наполняет базу данных.
-   `model_trainer.py`: Обучает модель и оценивает ее качество.
-   `recommender.py`: Генерирует управленческие рекомендации.
-   `inventory_forecast.db`: База данных проекта.
-   `demand_model.pkl`, `demand_scaler.pkl`: Артефакты обученной модели.
-   `model_performance.png`: Визуализация качества модели.
-   `procurement_report.txt`: Итоговый отчет по закупкам.

Прототип полностью функционален и соответствует всем пунктам технического задания.
